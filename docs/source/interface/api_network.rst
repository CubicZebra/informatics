_`Option networks`
==================

.. currentmodule:: info.docfunc

Description
-----------

Neural networks related tool-chain and conventional implementation via meta programming. For normal use, the
dependency of `PyTorch <https://pytorch.org/>`_ is required.

Namespace of this module is mainly in ``info.toolbox.networks``. For convenience in practice, use main
entry of ``info.net``.
